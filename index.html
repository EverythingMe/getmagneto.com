What is Magneto

Magneto allows you to write smart and powerful tests for Android apps.
It can be triggered from command line, IDE, CI servers and an interactive shell.
Compatible with Android 4.2+ apps.

Awesome Features

Log watching, skipping, tagging, blocking, screenshots, videos and even an interactive shell.

Open Source

Magneto is open source and free, developed with luv by EverythingMe. Come get involved!

Python Goodness

Magneto is Python and PyTest based, allowing test scenario syntax to be elegant and flexible.

Quick Start

Install 

$ pip install magneto

Bootstrap some tests 

$ magneto init calc

Run 'em 

$ magneto run
