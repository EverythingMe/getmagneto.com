What is Magneto

Magneto allows you to write smart and powerful tests for Android apps.
It can be triggered from command line, IDE, CI servers and an interactive shell.
Compatible with Android 4.2+ apps.

Quick Start

Install 

$ pip install magneto

Bootstrap some tests 

$ magneto init calc

Run 'em 

$ magneto run
