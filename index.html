What is Magneto

Magneto allows you to write smart and powerful tests for Android apps.
It can be triggered from command line, IDE, CI servers and an interactive shell.
Compatible with Android 4.2+ apps.

Magneto is open source and free, developed with luv by EverythingMe.

Quick Start

Install (requires pip) 

$ pip install magneto

Bootstrap some tests 

$ magneto bootstrap whatsapp

Run 'em 

$ magneto /magneto_tests

Basic Test Example

Here's a simple use case.
Click button, scroll list to bottom and check that last item exists. 

from magneto.base import BaseTestCase
from magneto.utils.assertion import Assert

class TestApp(BaseTestCase):
   def test_click(self):
       # click button
       button = self.magneto(text='Click me')
       button.click()

       # scroll list to bottom
       list = self.magneto(className='android.widget.ListView')
       list.fling.toEnd()

       # make sure last item exists
       last_item = self.magneto(text="last item")
       Assert.true(last_item.exists)

Finding an element can be done by text, className, description, etc.
